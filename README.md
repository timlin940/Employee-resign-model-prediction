# 員工離職預測模型 (Employee Resign Model Prediction)

## 📋 專案概述

本專案為一個 AI 機器學習模型，用於預測員工是否會離職。透過分析員工的相關特徵和歷史數據，建立預測模型，幫助企業提早識別高離職風險的員工，以便採取相應的人力資源措施。

## 🎯 專案目標

- 建立準確的員工離職預測模型
- 識別影響員工離職的關鍵因素
- 協助人資部門進行人才保留策略規劃
- 提供數據驅動的商業決策支持

## 📊 技術棧

- **程式語言**: Python
- **機器學習框架**: scikit-learn、TensorFlow 或 PyTorch
- **數據處理**: Pandas、NumPy
- **可視化**: Matplotlib、Seaborn
- **環境**: Jupyter Notebook
